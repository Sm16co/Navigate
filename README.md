# Navigate
Vid skapandet av menyerna har olika färger används. Dessutom har olika finesser används för att göra så att den understrukna texten som man hovar över ska ändra det understrukna mönstret. Dessutom har media queires används för att forma om menyn från en lodrätt till en horisontell ifall den enheten man använder är större än 450px.
